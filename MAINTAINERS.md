# Maintainers

## Repository Role

This repository is an independently maintained open source project.

It originated from the original `datart` codebase and is currently maintained as an independent project focused on buildability, security, dependency health, and controlled modernization.

## Maintainer Responsibilities

- review and merge maintenance changes
- keep the project buildable on supported toolchains
- document modernization decisions and tradeoffs
- coordinate releases and changelog updates
- preserve Apache 2.0 attribution and compliance

## Decision Policy

For substantial changes, prefer:

- explicit migration notes
- reversible rollout where practical
- focused verification commands
- compatibility-first upgrades before broad rewrites
