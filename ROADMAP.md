# Roadmap

## Near Term

- keep `JDK 21 + Spring Boot 3 + Node 24 + Vite 5` baseline stable
- continue reducing legacy Jest compatibility surfaces
- continue staged migration from Shiro runtime entrypoints toward Spring Security
- finalize rich text modernization path
- continue shrinking remaining outdated dependency and runtime hotspots

## Mid Term

- complete a single clear frontend test-stack direction
- upgrade lint and formatting toolchain to newer stable major lines
- continue Calcite and code-generation risk assessment
- improve release documentation and operational guidance for this independent project

## Long Term

- sustain a release line that is fully independent of the abandoned upstream cadence
- publish clearer release notes and compatibility matrix
- continue aligning repository, release, and product-facing materials under the `yu-bi` brand
